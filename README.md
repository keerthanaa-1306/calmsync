### Project Description
CalmSync ‚Äì A web-based mental wellness app offering guided breathing, mood tracking, calming activities, and supportive AI chat to help users manage stress and regain emotional balance.

### The Problem statement
In today‚Äôs fast-paced and demanding world, many people struggle with stress, anxiety, frustration, or sadness without having immediate access to supportive, non-judgmental tools to help them cope. When emotions go unprocessed, they can build up and negatively affect overall well-being. CalmSync bridges this gap with a simple, user-friendly web application designed for real-time emotional support. It guides users through emotional check-ins, offers calming activities such as breathing exercises and relaxing games, and provides supportive AI interactions to help individuals reset, regain clarity, and build emotional resilience whenever they need it.


### The Solution
CalmSync addresses the lack of accessible emotional support by offering a structured, interactive web application that helps users process their feelings in a private and guided way. The platform follows a progressive flow, beginning with a warm welcome, moving into mood check-ins, and providing personalized validation messages. Users can then choose supportive activities such as guided breathing exercises, calming games, AI-powered conversations, or symbolic ‚Äúburn it away‚Äù interactions to release negative thoughts. Built with modern web technologies like HTML, CSS, and JavaScript, and designed with an intuitive user experience, CalmSync delivers instant, stigma-free access to mental wellness tools directly through a browser‚Äîno downloads or subscriptions required. By lowering barriers and encouraging consistent engagement, the app supports emotional regulation and fosters long-term mental resilience through repeated, positive interactions.

## Features

List the key features of your project:

- Feature 1: Mood Check-In: Users select their current emotional state from options like Angry, Frustrated, Neutral, Sad, Anxious, or Stressed, with an optional text note.
- Feature 2: Emotional Validation: Personalized messages acknowledging the user's feelings, followed by activity choices.
- Feature 3: Guided Breathing Exercise: A visual breathing guide with inhale/exhale instructions and a countdown timer for mindfulness.
- Feature 4: Simulated AI Chat: Interactive conversation feature with pre-defined supportive responses to provide emotional support.
- Feature 5: Calming Game: An engaging, interactive game designed to distract and relax the user.
- Feature 6: Burn It Away Activity: A symbolic release mechanism where users can "burn" away negative thoughts through animated text interactions.
- Feature 7: Support Modal: Accessible information on mental health resources and app usage.
- Feature 8: Privacy-Focused: Fully client-side with no personal data collection or storage
